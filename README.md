# Playpaca Frontend

A React + Vite frontend application for Playpaca.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

## Tech Stack

- [React 19](https://react.dev/)
- [Vite 8](https://vite.dev/)
- [ESLint](https://eslint.org/)
